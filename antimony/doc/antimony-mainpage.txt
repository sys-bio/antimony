/** @mainpage Antimony: A modular human-readable, human-writeable model definition language

<P>Antimony is a text-based model definition language based on <A HREF="http://www.sys-bio.org/software/jarnac.htm">Jarnac</a>, and extended to be fully modular.  Antimony models can be converted to and from SBML, flattening the modularity in the process.  See also: @subpage antimony-why </P>

<h2>Downloads</h2>
<em>All users:</em>
- <A class="el" target="_top" HREF="https://sourceforge.net/projects/antimony/files/Windows%20binaries/1.3.0/antimonyInstaller_v1.3.exe/download">Windows Antimony installer</a> - includes QTAntimony (the GUI editor and SBML translator), as well as the command-line translators sbml2antimony.exe and antimony2sbml.exe
- <A class="el" target="_top" HREF="https://sourceforge.net/projects/antimony/files/Mac%20binaries/1.3.0/QTAntimony_v1.3.dmg/download">QTAntimony for Mac OSX</a> - the Antimony GUI editor and SBML translator, compiled for Mac OSX.
- This <A class="el" target="_top" HREF="http://downloads.sourceforge.net/antimony/Antimony_documentation_v1.3.zip">documentation</a>.

<em>Developers:</em>
- <A class="el" target="_top" HREF="http://downloads.sourceforge.net/antimony/antimony_src_v1.3.tar.gz">Source code</a> - a full package including the source code for libAntimony and QTAntimony as well as all documentation.  (You will also need <A target="_top" HREF="http://sbml.org/Software/libSBML">libSBML</a> v3.3.2 or higher for full functionality.)
- <A class="el" target="_top" HREF="http://downloads.sourceforge.net/antimony/Antimony_v1.3.zip">Antimony Windows Developer Package</a> - For developers: the antimony.dll and associated files and binaries.
- <A class="el" target="_top" HREF="https://sourceforge.net/project/showfiles.php?group_id=251883">Other Downloads</a> - The above packages as well as previous versions.

<h2>Information</h2>
<em>All users:</em>
- The <A class="el" HREF="Tutorial.pdf">Tutorial</a> will walk you through creating Antimony files from scratch.
- @subpage antimony-qt (compiled for Windows, MacOS, and Linux)
- @subpage antimony-examples are provided to show some of Antimony's features.  Included are the 211 @subpage antimony-biomodels (from the originals at <A target="_top" HREF="http://biomodels.net/">BioModels.net</a>).
- Our <A class="el" target="_top" HREF="http://bioinformatics.oxfordjournals.org/cgi/content/short/25/18/2452">Bioinformatics</a> article.

<em>Developers:</em>
- The @subpage antimony-technical-spec is a reference manual, explicitly describing how to parse Antimony models.
- @subpage antimony-installation for installing and compiling libAntimony.
- The <A class="el" HREF="antimony__api_8h.html">available API commands</a> when incorporating libAntimony into your own projects.
- For full functionality, libAntimony makes use of <A target="_top" HREF="http://sbml.org/Software/libSBML">libSBML</a> to convert models to and from the SBML format.
